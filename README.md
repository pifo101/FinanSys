# FinanSys

## Live Demo

https://finansys.lat

## 🇺🇸 English

FinanSys is a web-based financial calculator built with Flask. It provides tools for loan amortization, flexible financial calculations, and late payment interest calculations.

### Features

- Loan amortization calculator with monthly payment schedule.
- Flexible calculator to determine the initial amount, final amount, interest rate, or time period.
- Late payment interest calculator with configurable grace period.
- Informational pages describing the project.

### Requirements

- Python 3.10 or later
- pip

## Technologies

- Python
- Flask
- HTML5
- CSS3
- JavaScript

### Installation

Clone the repository:

```bash
git clone https://github.com/pifo101/FinanSys.git
cd FinanSys
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate the virtual environment (Windows):

```bash
.venv\Scripts\activate
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

### Running the application

Start the application:

```bash
python app.py
```

Then open your browser and visit:

```text
http://127.0.0.1:5000
```

### Project Structure

- `app.py` – Main Flask application and routes.
- `logica.py` – Financial calculation logic.
- `templates/` – HTML templates.
- `static/` – CSS, JavaScript, and image assets.
- `requirements.txt` – Project dependencies.
